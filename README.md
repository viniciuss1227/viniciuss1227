## 🚀 Projetos

### [SRS Flashcards](https://github.com/viniciuss1227/srs-project)
API fullstack para gerenciamento de flashcards com **algoritmo customizado de repetição espaçada (SRS)**.  
Calcula dinamicamente o próximo intervalo de revisão por card com quatro níveis de confiança (Again, Hard, Good, Easy) — diferenciando o projeto de um CRUD convencional.

**Stack:** Python · FastAPI · SQLAlchemy · SQLite · JavaScript Vanilla · HTML5 · CSS3

---

### [Resumidor de Texto com IA](https://github.com/viniciuss1227/resumidor-texto-ia-php)
Aplicação web em PHP que consome uma **API de IA (APYHub)** via cURL para gerar resumos automáticos de textos em PT-BR, com histórico das últimas interações persistido em MySQL usando **prepared statements**.

**Stack:** PHP · MySQL · cURL · HTML5 · CSS3
